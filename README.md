# Uptime Monitor

A simple uptime monitoring application built with FastAPI and Jinja2.

## Features

- **Real-time Monitoring** - Monitor your websites 24/7
- **Dashboard** - Beautiful dashboard with live status updates
- **Response Times** - Track average latency and response times
- **Uptime History** - Visual timeline of uptime with color-coded status blocks
- **User Authentication** - Secure login and user profiles
- **Multiple Websites** - Monitor multiple websites simultaneously
- **Status Indicators** - Green for up, red for down, clear status badges

## Installation

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Visit `http://localhost:8000`
